libxio is a C++ socket toolkit, it contains many useful classes for creating network software. It's cross-platform and released under the MIT License.

## Platforms

* iOS (Xcode 8+)
* Android (NDK r10e+)
* macOS (Xcode 8+)
* Linux (GCC 5+)
* Windows (VS 2015+)
